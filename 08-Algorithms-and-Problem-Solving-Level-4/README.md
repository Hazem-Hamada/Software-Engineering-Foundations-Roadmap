## 🧠 The "System-Level" Mindset
In this stage, the focus shifted from "writing functions" to **"engineering systems."** I applied advanced C++ concepts to manage large-scale logic and data integrity.

**Key Learnings:**
* **Time & Date Engineering:** Building leap year logic, custom calendars, and date arithmetic from scratch without external libraries.
* **Granular Security:** Implementing **Bitwise Operations** to manage user permissions efficiently in memory.
* **Session Management:** Tracking active states (Current User/Client) to ensure secure and isolated transaction environments.
* **Endurance & Scale:** Solving 65+ new problems to internalize complex patterns and increase analytical speed.

---

## 🛠️ Technical Skills Demonstrated
* **Bitwise Permission Handling:** Efficiently managing multi-flag access levels.
* **Advanced File Synchronization:** Maintaining consistent state between memory (Vectors) and disk (Flat Files) during complex CRUD operations.
* **Complex Logic Architecture:** Handling long-form problems that require "Divide & Conquer" at a higher level of nesting.
* **Defensive Programming:** Enforcing strict business rules (e.g., balance limits, PIN validation, date range overlaps).
