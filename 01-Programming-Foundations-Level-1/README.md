# 📘 Programming Foundations – Level 1
> **"Build the Mental Model Before You Write Code"**

This repository contains my progress and notes for the first course in my software engineering journey. The primary goal was to understand how computers think and process data before touching any specific programming syntax.

---

## 🗝️ Course Overview
Instead of rushing into coding, this course focuses on the fundamental "under the hood" operations. It builds a solid conceptual foundation that makes learning any programming language (C++, C#, Python, etc.) significantly easier and deeper.

## 📖 Key Topics Covered
* **Data vs. Information:** Understanding the hierarchy from raw data to wisdom.
* **Number Systems:** Deep dive into **Binary, Hexadecimal, and Octal** systems and how to convert between them.
* **Computer Architecture:** How internal components process data and the concepts of memory (Bits, Bytes, Nibbles).
* **Translation Process:** The role of Compilers, Interpreters, Assemblers, and Linkers.
* **Programming Logic:** Understanding variables, constants, and operator precedence.

## 🎯 Learning Outcomes
* Developed a "Programmer's Mental Model."
* Ability to reason about how high-level code eventually runs on hardware.
* Prepared to tackle Problem Solving and Algorithms with a clear mind.
