# 📘 Introduction to Programming with C++ – Level 1
> **"From Thinking to Real Programming — My First True Programming Language"**

This repository marks the transition from pure logic to technical implementation. After building a solid mental model and mastering algorithmic thinking, I started using **C++** as a powerful tool to bring those logic-based solutions to life.

---

## 🗝️ Course Overview
This course isn't just about learning C++ syntax; it's about learning how to express correct thinking through code. I focused on building a deep understanding of how C++ works and why it’s a fundamental language for any serious software engineer.

## 📖 Key Topics Covered
* **C++ Syntax & Environment:** Setting up a professional development environment and writing structured code.
* **Data Handling:** Working with various Data Types, Variables, and Constants.
* **Operators & Expressions:** Mastering arithmetic, logical, and relational operators.
* **Control Flow:** Implementing decision-making (Conditions) and repetition (Loops).
* **Code Reusability:** Introduction to **Functions** and scope.
* **Data Structures:** Working with Arrays, Structures, and Enums to organize data.
* **Memory Basics:** A beginner-level look at how C++ interacts with memory.

## 🛠️ Practical Implementation
During this course, I translated the **50 logical problems** previously solved in the "Algorithms" course into actual C++ programs. This phase was about getting familiar with the syntax and ensuring the logic remains sound when executed by a machine.
